# memoake

一度起動すれば、コマンド一発でアプリが表示/非表示ができるメモアプリ

## 解決したい問題

- ちょっとしたメモをしたいときに手軽にローカル上にメモができる
  - コマンドは最低限で他よりも最速でメモができる
- メモをファイルシステム上に配置するのは管理が大変（どこに保存したか忘れる）のでファイルシステムに頼らない管理
  - ファイル名を付ける必要がない -> プロジェクト名も付けるのが面倒かもしれない（なにか良い方法はないか）
  - 全文検索などでファイルよりも速く検索することができる

## 開発方針

- yakuakeスタイルの高速メモ
- ファイル構造（管理）やタイトルなどを意識させずにメモ本文を書くことだけに集中できる
- シンプルで機能は限定的にしてメモリやディスクを節約する

## 機能

- メモをプロジェクト単位で管理
- メモを保存、編集、削除ができる

## 課題

- 基本的に英語で記載
- 起動した時点のディスプレイに固定される -> カーソルがあるディスプレイに表示したい
- 全文検索などもできれば実装したい
